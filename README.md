# CS-320
# How can I ensure that my code, program, or software is functional and secure?
# By adhering to industry coding standards, following technical specifications, and performing tests to ensure my code works as intended.
#
# How do I interpret user needs and incorporate them into a program?
# Interpretting user needs requires coordination with users and proper comprehension and integration of user's needs into the requirements document.
# Feedback from the user is needed to ensure that you are creating what was intended and that you actually meet all of their requirements.
#
# How do I approach designing software?
# Careful analysis of the technical requirements while incorporating adequate testing procedures to verify code response.
